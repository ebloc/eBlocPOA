# POA

`Geth` version should be greater or equal than 1.7.3.

Please update `DATADIR` variable on `client.sh` and `server.sh` as your path for ebloc_POA directory.


## Initializing

```
geth --datadir ~/ebloc_POA/private init custom.json
```

## Server run:

```
sudo bash server.sh
```

## Client run:

```
sudo bash client.sh
```
